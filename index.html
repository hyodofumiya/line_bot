勤怠管理アプリケーション

内容

LINEアカウントと連携した勤怠管理アプリケーションです。
LINEアプリを使用することで、様々な入力を簡易化することができます。

特徴

  1. スムーズなUXを提供

    • LINEアプリ上で、ほぼ全ての機能を利用可能
    • 使用頻度の高い処理は１タップで実行
    • ログイン不要（LINEアプリ版）
    • アカウント連携でログイン時の手間を削減（ブラウザ版）

  2. リモートワーク×フレックスタイム制を想定した仕様

    勤務時間中に連絡できるよう、ステータスを設定（作業中・休憩中・退勤中）

担当領域

企画・要件定義・設計・コーディング・簡易テスト・リリースまでの全領域

開発環境

  • OS : MacOS
  • 言語 : Ruby version 2.5.1
  • FW : Ruby on Rails version 5.2.4
  • DB : PostgreSQL

工夫・こだわり

  • UX

    ①アクション数、②ツール、③ユーザー教育の削減

    →使用頻度の多い処理は１タップで実行

    →UIは、普段から使い慣れているLINEを選択

    →誰でも使いこなせるように、多機能ではなく必要十分となるように設計

  • 今後のニーズを重視した仕様

    リモート×フレックスタイム制の勤務形態を想定して設計。

    →作業中・休憩中・退勤中のステータスを管理者が確認できるようにし、プライベートタイムに干渉しないようにした。

使い方

  • 非管理者ユーザーの場合

      LINEアプリで利用

    1. LINEアプリで＿を友達追加する
    2. メッセージに従いユーザーを登録する
    3. リッチメニューから各機能を使用する
      各リッチメニューボタンの説明
      出勤 出勤時に使用します。ステータスが作業中になります。
      休憩開始 休憩開始時に使用します。ステータスが休憩中になります。
      休憩終了 休憩終了時に使用します。ステータスが作業中に戻ります。
      退勤 退勤時に使用します。ステータスが退勤中になります。休憩中でも自動で休憩終了し、退勤します。
      今月と先月 今月と前月の勤務履歴を一覧で表示します。
      日付を指定 日付を指定して、勤務履歴の詳細を表示します。
      修正 勤怠履歴を修正します。
      報連相 未実装です。社員間でステータスの確認、管理者への連絡機能を想定。

      ブラウザで利用

    1. LINEアプリで＿を友達追加する
    2. メッセージに従いユーザーを登録する
    3. ログイン画面を開く
    4. 社員登録したLINEアカウントでLINEログイン
    5. ログイン後は全ての機能を利用できます。
    6. 使い終わったら下記管理者ユーザーで自身の社員情報を削除してください。

  • 管理者ユーザーの場合

    • 管理者用ログイン画面から下記を入力してログインします。

      　 E-mail: admin@sample.com

      　 Password: adminuser


    • ログイン後は全ての機能を利用できます

  • その他

    • 検索機能について

      社員情報は、社員番号,セイ,メイで検索できます。

      勤怠履歴は、社員番号,日付で検索できます。


    • メッセージ送信機能について

      メッセージは、自分で作成した以外のユーザーには送信しないでください。
