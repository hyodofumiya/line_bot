内容

LINEアカウントと連携した勤怠管理アプリケーションです。
LINEアプリを使用することで、様々な入力を簡易化することができます。

特徴

  • スムーズなUXを提供

    • LINEアプリ上で、ほぼ全ての機能を利用可能
    • 使用頻度の高い処理は１タップで実行
    • ログイン不要（LINEアプリ版）
    • アカウント連携でログイン時の手間を削減（ブラウザ版）

  • リモートワーク×フレックスタイム制を想定した仕様

    出退状況に加え、作業中か休憩中か確認できる。

担当領域

企画・要件定義・設計・コーディング・簡易テスト・リリースまでの全領域

開発環境

OS MacOS
言語 Ruby version 2.5.1
FW Ruby on Rails version 5.2.4
DB PostgreSQL
サーバー Heroku
